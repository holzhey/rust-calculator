### Calculator

[![dependency status](https://deps.rs/repo/github/holzhey/rust-calculator/status.svg)](https://deps.rs/repo/github/holzhey/rust-calculator) [![maintenance-status: experimental](https://img.shields.io/badge/maintenance--status-experimental-palevioletred)](https://gist.github.com/rusty-snake/574a91f1df9f97ec77ca308d6d731e29)

A simple POC for now, where i implement a very limited sum only calculator using HTMX.

When feasible i will add more operations and a better UX.

The goal is to proof that HTMX can be used in a clean way, having everything handled by a RUST+AXUM+MAUD backend.
